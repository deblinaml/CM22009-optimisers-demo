📚 Key Differences

Full-Batch Gradient Descent

Uses all training data to compute the gradient at each step.

φ ← φ - α·Σᵢ ∂ℓᵢ/∂φ

✓ Smooth, deterministic path
✗ Can get stuck in local minima
✗ Expensive for large datasets

Stochastic Gradient Descent

Uses a random mini-batch of data at each step.

φ ← φ - α·Σᵢ∈Bₜ ∂ℓᵢ/∂φ

✓ Noise helps escape local minima
✓ Computationally efficient
✓ Often finds better solutions
