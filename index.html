How to Use These Visualizations

  • Open these in full screen. Use the controls to show what happens with different parameters.
  • Student Self-Study: PLay with the different parameters. Try edge cases.
  • Key Experiments: Try very high/low learning rates, compare behaviour on convex vs non-convex functions, observe how momentum helps in narrow valleys (You can try momentum after the next Wednesday lecture).
  • Discussion Points: Why does SGD sometimes find better solutions? When does Adam outperform momentum? What makes a function hard to optimize?
